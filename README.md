# Articles and comments
* Basic ruby blog with simple authentication.
* Followed this guide: https://guides.rubyonrails.org/getting_started.html

### Configuration
```
git clone {repo}
cd {repo}
bundle install
```

### Database creation
`rake db:create`

### Database initialization
`rake db:migrate`

### Ruby version
2.7.0

### System dependencies
Postgres 12
